Command Line Interface

Perfect for automated scripts, batch processing, and integration with existing workflows:

# Basic usage
python3 cli.py scrape --funds "Citadel Advisors" "AQR Capital Management"

# First-time filers only
python3 cli.py scrape --funds "New Fund LLC" --only-first-time

# Holdings filtering
python3 cli.py scrape --funds "Large Fund" --min-holdings 100
python3 cli.py scrape --funds "Small Fund" --between-holdings 10 50

# Batch processing
python3 cli.py scrape --funds-csv sample_funds.csv --quarter 2024Q4

Python API

Integrate directly into your Python applications and data pipelines:

from logic import ThirteenFProcessor

with ThirteenFProcessor() as processor:
    results = processor.process_funds(
        funds=["Citadel Advisors"],
        quarter="2024Q4",
        min_holdings=100
    )
    
    for result in results:
        print(f"{result.fund_name}: {result.num_holdings} holdings")
        print(f"First-time filer: {result.is_first_time_filer}")
        print(f"Filing URL: {result.filing_url}")

üîß Technical Architecture

Data Sources & APIs

  • SEC Company Submissions API - For filing history and metadata retrieval
  • EDGAR Full-Text Search - For document retrieval and parsing
  • Information Tables - XML/TXT parsing for detailed holdings data
  • Rate Limiting - Respects SEC guidelines with intelligent throttling

Output & Reports

The tool generates comprehensive, professional-grade reports:

  • Summary Report - Overview of all processed filings with key compliance metrics
  • Holdings Data - Detailed holdings per fund in CSV format for Excel analysis
  • JSON Lines - Machine-readable format for data pipelines and APIs
  • Metadata - Filing URLs, dates, and first-time filer status for audit trails

Enterprise Features

  • Error Handling - Graceful fallback and comprehensive error reporting
  • Data Validation - Robust validation and cleaning of all parsed data
  • Caching System - Intelligent caching to reduce redundant API calls
  • Logging - Comprehensive logging for compliance and debugging
